# evidencias_reto

En este repositorio se estarán almacenando los entregables de las evidencias a realizar. Incluyendo archivos de documentación y ejecución.

# Diagrama de Gantt
Para visualizar el diagrama de Gantt, favor de descargar el archivo Gantt.json para luego cargarlo en el visualizador. Favor de cargar el archivo anteriormente descargado en el [visualizador de Gantt](https://carleslc.me/Gantt/) para más detalle.
